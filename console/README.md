# MFOS Commit Boundary Console

Single-page React + TypeScript UI for running MFOS middleware decisions against cases in `fixtures/mfos_ground_truth_v1_4_1.json`.

## Run

```bash
cd console
npm install
npm run dev
```

Open the URL shown by Vite (usually `http://localhost:5173`).

## Build

```bash
cd console
npm run build
npm run preview
```
